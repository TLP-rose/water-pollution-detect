# 项目文档目录
本目录用于统一存放项目过程中的规则文档、工作日志及其他项目文档材料。

### 目录结构（更新中 4.13）

```text
docs/
├─ team_rules/   # 团队协作规则、文档规范、分工说明等
├─ work_logs/    # 团队成员按周整理的工作日志
└─ README.md     # 当前目录说明文件
```

### 使用要求
1. 规则类文档统一放在 team_rules/ 
2. 工作日志统一放在 work_logs/ 
3. 不要将个人临时文件、无关文件随意放入 docs/ 
4. 新增文档前先确认所属目录，保持目录整洁 
5. 如后续增加新的文档类型，可再单独建立子目录

### 说明
本目录主要服务于项目过程管理、竞赛材料整理和后续答辩复盘。
